\# cypress-makishop-e2e



Suite de automatización E2E para \*\*MAKISHOP\*\*, tienda tecnológica basada en Odoo.



\## Descripción



Proyecto de automatización de pruebas funcionales sobre el sistema ERP Odoo,

simulando flujos reales de una tienda de tecnología: login, clientes, ventas,

facturas, compras, CRM, empleados e inventario.
